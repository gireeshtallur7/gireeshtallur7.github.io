📁 Projects Included
1. Telecom Customer Churn Prediction
Machine Learning Classification

Built a predictive model to identify customers at high risk of churning. Used customer demographics, account details, and usage patterns to help telecom companies proactively retain subscribers.

Tech: Python, Scikit-learn, Pandas, Matplotlib

Algorithms: Logistic Regression, Random Forest, XGBoost

Key Outcome: Achieved 85%+ recall in identifying churn-risk customers

2. Heart Disease Prediction Using ML
Binary Classification for Healthcare

Developed a machine learning model to predict the likelihood of heart disease based on clinical parameters like age, cholesterol, blood pressure, and chest pain type.

Tech: Python, Scikit-learn, Seaborn, NumPy

Algorithms: KNN, SVM, Decision Tree, Random Forest

Key Outcome: Random Forest achieved 88% accuracy with balanced precision/recall

Approach: TF-IDF + Logistic Regression / LSTM (mention if used)

Key Outcome: 90% accuracy on test dataset
